Setting up your site

This extension uses the old flash cooliris.swf file hosted by the cooliris team.

For this extension to work your site will need;

  • a crossdomain.xml
  • a thumbnail.rss

Crossdomain xml

Your crossdomain.xml should be in the root of your website e.g. www.mysite.com/crossdomain.xml. and look like this...


<?xml version="1.0"?>
<!DOCTYPE cross-domain-policy SYSTEM
    "http://www.macromedia.com/xml/dtds/cross-domain-policy.dtd">
<cross-domain-policy>
    <allow-access-from domain="*.cooliris.com" secure="false" />
</cross-domain-policy>

For more information see;

  • www.cooliris.com/developer/reference/media-rss
  • www.adobe.com/devnet/articles/crossdomain_policy

Thumbnail RSS

You need to link to a valid media RSS feed in your web page.

To set up a valid media RSS feed see www.cooliris.com/developer/reference/media-rss
